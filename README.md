# PLINKO∞ — Idle Plinko Game

Standalone idle Plinko game. **Its own project — not under net app or Raven.**

- **Preview (phone frame):** open `index.html` in a browser.
- **Game full screen:** open `game.html` in a browser.
- **Dev server:** from this folder run `npm install` then `npm run dev` (if using Vite config).

Run and build from this folder only.
